# Dataset
Plese see infolog.txt to see the datasets in the file.
The dataset name, shape and datatype are shown.
This file is a timestamp file, containing precipitation data all over 1800 Latitudes *  3600 Longitudes over the world.
The x_bnds datasets give the bounded interval of the variable x. 
For eg lon_bnds goves bounds of longitude: [[-180. to -179.9][-179.90001 to -179.8]] which are represented by their midpoint in the lon dataset.
